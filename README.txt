En el repositorio se encuentra el proyecto en React, la api y la base de datos

Para instalar la base de datos debe crear una nueva (recomendable usar phpMyAdmin) y después exportar el archivo "konecta-coffee" que se encuenra en la carpeta "dataBase".

una vez clonado el repositoro, debe abrir una terminal dentro del proyecto y ejecutar el comando npm start, este iniciará el aplicativo web
